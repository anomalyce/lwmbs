# lwmbs

一个 ^ 名字很奇怪的micro/cli构建系统，理论上不会有很多人看到这个项目所以无所谓了

# 用法

```bash
# optional
export https_proxy=https://someproxy
export GITHUB_USER=someuser
export GITHUB_TOKEN=ghp_dsad
mkdir build
cd build
# prepare sources
../update_source.php some.json
../fetch_source.php some.json
# build micro
../build_micro.php
```
